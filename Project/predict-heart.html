Heart Failure Prediction

The primary objective of this project is to build an accurate classification model to predict heart failure. The model was trained using a dataset from the UCI Repository, which contains information such as platelet count, blood pressure status, creatine kinase levels, and other variables. This project demonstrates an end-to-end workflow for building and deploying a machine learning model.
